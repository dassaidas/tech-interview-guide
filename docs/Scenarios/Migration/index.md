## 1. Legacy system assessment

### 1. What is the role of Legacy system assessment in legacy migration scenarios?

**Answer:**

In legacy migration scenarios, the term Legacy system assessment refers to the discovery work used to
understand current architecture, risk, dependencies, and business criticality. It is part of the
foundation a candidate should be able to explain clearly.

```yaml

  phases:

    - cutover
```

Success metrics connects to the rest of legacy migration scenarios by giving structure to the
measurements used to decide whether migration outcomes are actually improving the system. It is one
of the pieces that turns isolated facts into a coherent end-to-end explanation.

**Sample:**

```yaml
migration:
  concept: "10. Success metrics"
  phases:
    - assess
    - extract
    - validate
    - cutover
```
