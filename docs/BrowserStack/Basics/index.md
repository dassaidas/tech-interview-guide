# BrowserStack Basics Interview Questions

![BrowserStack Basics Interview Questions](../../assets/browserstack-testing-flow.svg)

This page focuses on BrowserStack as a practical cross-browser and cross-device testing platform.

## 1. BrowserStack Live

### 1. What is the role of BrowserStack Live in BrowserStack basics?

**Answer:**

In BrowserStack basics, the term BrowserStack Live refers to the manual browser and device testing experience
used for interactive validation. It is part of the foundation a candidate should be able to explain
clearly.

**Sample:**

```yaml
# Concept: 1. BrowserStack Live
browserstack:
  username: ${BROWSERSTACK_USERNAME}
  accessKey: ${BROWSERSTACK_ACCESS_KEY}
  buildName: sample-build
  projectName: 1. BrowserStack Live
```
